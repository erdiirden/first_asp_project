## HTTP Fonksiyonları
* Get: filtreleme, listeleme gösterme vs.
* Post: giriş yapma, ekleme yapma vs.
* Put: güncelleme yapma
* Patch: local güncelleme yapma (ufak değişikliklerde tercih edilir)
* Delet: silme işlemleri
